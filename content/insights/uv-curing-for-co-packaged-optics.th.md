## เหตุใดขั้นตอนการบ่มจึงสำคัญ

ใน CPO หรือ Silicon Photonics การเปลี่ยนตำแหน่งเพียงเล็กน้อยอาจส่งผลต่อประสิทธิภาพการเชื่อมต่อแสง การหดตัวของกาว การฉาย UV ไม่สม่ำเสมอ การคลายตัวของ Fixture หรือการเปลี่ยนแปลงอุณหภูมิอาจทำให้ชิ้นส่วนเคลื่อน ค่าความคลาดเคลื่อนต้องอ้างอิงจากอุปกรณ์ อินเทอร์เฟซ และแผนการทดสอบของแต่ละโครงการ ไม่ควรใช้ตัวเลขเดียวกับทุกแบบ


## 1. ควบคุมการหดตัวและความเค้น

- ควบคุมปริมาณกาวและความหนาของ Bond Line
- ฉาย UV แบบสมมาตรเมื่อทำได้
- หลีกเลี่ยง UV Dose สูงเฉพาะจุด
- ยืนยันว่าต้องมี Thermal หรือ UV Post-Cure หรือไม่
- วัดการเปลี่ยนตำแหน่งระหว่างการพัฒนากระบวนการ

## 2. รักษา Active Alignment

ลำดับการบ่มต้องสนับสนุนสถานี Active Alignment และไม่รบกวนตำแหน่งที่เหมาะสม อาจใช้การฉายสั้นเพื่อยึดชิ้นส่วนก่อน แล้วจึงบ่มให้สมบูรณ์แบบควบคุมตามชนิดกาว การเข้าถึงของแสง Fixture และ Takt Time


## 3. ส่งมอบ UV Dose ที่ทำซ้ำได้

- วัดเอาต์พุตที่ตำแหน่งผลิตจริง
- กำหนดช่วง Irradiance และ UV Dose ที่ยอมรับได้
- บันทึกค่าตั้งและสถานะการสอบเทียบ
- ติดตามสภาพ Light Guide หรือ LED Head
- ตรวจสอบใหม่หลังการเปลี่ยนแปลงทางออปติคัลหรือกลไก

## 4. เลือกสถาปัตยกรรมแหล่งกำเนิด UV

ระบบ UV LED Spot แบบหลายช่องสามารถรองรับหลายจุดหรือการฉายตามลำดับ ส่วนระบบ Spot แบบใช้หลอดอาจเหมาะกับงานพัฒนา การรับรอง หรือกระบวนการที่ต้องใช้สเปกตรัมกว้างกว่า ควรเลือกตามการตอบสนองของกาว จำนวนจุด พื้นที่ Cycle Time การตรวจสอบย้อนกลับ และกลยุทธ์การบำรุงรักษา


## 5. สร้างหลักฐานคุณภาพในกระบวนการ

ในการพัฒนาผลิตภัณฑ์ใหม่และการผลิตจำนวนมาก ควรบันทึกแหล่ง UV ความยาวคลื่น Irradiance เวลา UV Dose ที่คำนวณ สถานะการสอบเทียบ และการเตือนที่เกี่ยวข้อง ข้อมูลเหล่านี้ช่วยแก้ปัญหาและควบคุมการเปลี่ยนแปลง แต่ไม่แทนที่การทดสอบด้านออปติคัลและความน่าเชื่อถือ


## ขั้นตอนพัฒนาที่แนะนำ

กำหนดข้อกำหนดการจัดแนวและความน่าเชื่อถือ

คัดเลือกกาวร่วมกับผู้ผลิตวัสดุ

ยืนยันความเข้ากันได้ของสเปกตรัมและการเข้าถึงของแสง

พัฒนาการยึดเบื้องต้นและการบ่มสุดท้ายที่มีความเค้นต่ำ

วัดการเคลื่อนตำแหน่งและประสิทธิภาพการเชื่อมต่อแสง

กำหนด Process Window ที่ผ่านการยืนยัน

ถ่ายโอนค่าตั้ง เกณฑ์ตรวจสอบ และแผนบำรุงรักษาสู่การผลิต


## หลีกเลี่ยงการใช้ข้อมูลโครงการเดียวเป็นมาตรฐานทั่วไป

สถาปัตยกรรม CPO มีความแตกต่างกันมาก ค่าความคลาดเคลื่อน เงื่อนไข Thermal Cycling และเวลาบ่มต้องมาจาก Drawing ข้อกำหนดอุปกรณ์ และแผน Qualification ของลูกค้า ไม่ควรนำตัวเลขของแบบหนึ่งมาใช้เป็นข้อกำหนดของ CPO ทั้งหมด

ETIA จัดจำหน่ายระบบและชิ้นส่วน UV ของแท้ผ่านช่องทางที่เป็นทางการ พร้อมสนับสนุนการเลือกแหล่งกำเนิด การทดลองกระบวนการ การทบทวนการกำหนดค่า และบริการด้านเทคนิคในพื้นที่.
