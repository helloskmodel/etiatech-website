## 1. เริ่มจากข้อมูลของกาว UV

- ยืนยันยี่ห้อและรุ่นกาว
- ตรวจสอบช่วงคลื่น Irradiance และ UV Dose ที่แนะนำ
- ประเมินขีดจำกัดอุณหภูมิ ความหนาชั้นกาว และบริเวณเงา
- ยืนยันระยะทำงานและ Cycle Time
ความยาวคลื่นที่ตรงกันเพียงอย่างเดียวยังไม่ยืนยันการบ่มที่สมบูรณ์ ต้องตรวจสอบ Irradiance เวลา UV Dose และรูปทรงการประกอบด้วย


## 2. เลือกโมดูลหลอด

Standard Lamp Module 012-68000R เหมาะกับงานประกอบอิเล็กทรอนิกส์ อุปกรณ์การแพทย์ อุปกรณ์ออปติคัล และไฟเบอร์ออปติกหลายประเภท

Surface Cure Lamp Module 012-69000R อาจช่วยกาวที่รองรับเมื่อผิวยังเหนียวจาก Oxygen Inhibition แต่สาเหตุอาจมาจาก UV Dose ไม่พอ ช่วงคลื่นไม่เหมาะ ระยะทำงานมากเกินไป หรือ Light Guide สกปรกได้เช่นกัน


## 3. ตรวจสอบหมายเลขชิ้นส่วน

- 012-68000R — Standard Lamp Module
- 012-69000R — Surface Cure Lamp Module
- 012-64000R — หลอดอะไหล่ 200 W สำหรับชุดมาตรฐานที่รองรับ
- 012-54000R — หลอดอะไหล่สำหรับชุด Surface Cure ที่รองรับ

## 4. เลือกฟิลเตอร์ออปติคัล

- 019-00387R — 400–500 nm
- 019-00388R — 365 nm
- 019-00389R — 320–390 nm
- 019-00390R — 250–450 nm
- 019-00391R — 320–500 nm
- 019-00392R — Blank / ไม่มี Band-Pass Filter
เลือกช่วงคลื่นตามการดูดกลืนของกาวและผลการตรวจสอบกระบวนการ กำลังรวมที่สูงกว่าไม่ได้หมายความว่าการบ่มจะดีกว่า สิ่งสำคัญคือพลังงานที่กาวดูดกลืนได้จริงบริเวณยึดติด


## 5. ประเมินเส้นทางแสงทั้งหมด

Irradiance ที่ชิ้นงานยังขึ้นอยู่กับประเภทและขนาด Light Guide สภาพการใช้งาน จำนวนแขนง ระยะทำงาน ขนาดจุดฉาย การจัดตำแหน่ง และความสะอาดของชิ้นส่วนออปติคัล


## 6. ตรวจสอบหลังเปลี่ยนส่วนประกอบ

รอให้หลอดคงที่ตามคู่มือ

ติดตั้งฟิลเตอร์และ Light Guide ที่ใช้จริง

วัดด้วย Radiometer ที่รองรับและผ่านการสอบเทียบ

ยืนยัน Irradiance ที่ระยะทำงานจริง

ตรวจสอบเวลาและ UV Dose

ทดสอบกับชิ้นงานจริง

ETIA จัดจำหน่ายอุปกรณ์และชิ้นส่วนระบบบ่ม UV ของแท้ผ่านช่องทางที่เป็นทางการ พร้อมให้คำแนะนำด้านการใช้งาน การตรวจสอบการกำหนดค่า และการสนับสนุนทางเทคนิคในพื้นที่.
