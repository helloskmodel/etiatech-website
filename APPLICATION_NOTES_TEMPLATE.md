# Application Notes — 写作模板 / Writing Template

> 用途:写手按每个 ID 填写 `Challenge / Solution / Benefit / Recommended` 四段。
> 规则:**ID、Industry、Sub-category 不要改**(改了会对不上代码)。Title 可微调。
> 共 **10 大行业 / 57 个应用点**。带 ⭐ 的是当前首页/列表的「精选(hot)」案例,可保留或调整。
> 字段含义:
> - **Challenge** 痛点/工艺难题(2–4 句)
> - **Solution** ETIA 方案 + 用到的设备/波长/工艺(2–4 句,点名具体型号)
> - **Benefit** 量化收益(节拍、良率、合规、能耗等,尽量给数字)
> - **Recommended** 推荐型号组合(一行)
